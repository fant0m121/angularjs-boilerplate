# angularjs-boilerplate
[![Dependency Status][depstat-image]][depstat-url]

Empty project for angularjs

Install dependencies
```shell
npm install && bower install
```
Serve in dev mode, with hot reload at localhost:3000
```shell
gulp serve
```
